# Hermes Lite 2 OpenWebRX Front Page

This project offers a simple, accessible landing page that allows anyone to access your Hermes Lite 2 SDR receiver using OpenWebRX.

## Features

- Large "Listen Now" button linking to OpenWebRX.
- Basic instructions for first-time users.
- Placeholder for receiver status (can be enhanced if OpenWebRX exposes an API).
- Contact/info section.

## Setup & Deployment

1. **Edit `index.html`** with your personalized info or features.
2. **Push files to your GitHub repository.**
3. **Enable GitHub Pages:**  
   - Go to your repo’s Settings → Pages.
   - Set the source branch (usually `main` or `master`) and `/ (root)` folder.
   - Your page will be live at `https://<your-username>.github.io/<repo-name>/`

4. (Optional) Configure your custom domain (`vk2icw.com`) in the GitHub Pages settings.

## Customizing

- Update the instructions in `index.html` as needed.
- Modify `style.css` for color, layout, branding.
- Add JavaScript in `script.js` for advanced features (receiver status, user feedback).
